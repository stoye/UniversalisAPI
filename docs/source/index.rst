UniversalisAPI documentation
============================

**UniversalisAPI** is a Python library for asynchronously interacting with the public API of
`Universalis.app <https://universalis.app>`_. See :doc:`usage` for usage information, including
:ref:`installing <installation>` the project.

.. note::
   This project is under active development. Basic funtionality is present, but some features
   are not fully implemented.
